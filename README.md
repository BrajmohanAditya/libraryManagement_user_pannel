# 📚 Library Management System - Multi-Tenant Documentation

## Overview

A comprehensive multi-tenant library & learning management system with role-based access control. The platform supports **Super Admin**, **Org Admin**, **Teacher Admin**, and **User** roles across multiple applications.

## 🔌 APIs

### User API (Library Management)
- **Base URL**: [https://library-management-user-backend.onrender.com/api#](https://library-management-user-backend.onrender.com/api#)
- **Description**: Backend API for library user operations (borrow, return, search, etc.)

### Main LMS API (Multi-tenant)
- **Base URL**: [https://api.khansirlearning.com/api](https://api.khansirlearning.com/api)
- **Description**: Core API for organization-based learning management (courses, users, roles, tenants)
